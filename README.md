 FastAPI Food Delivery App

This project is a backend system built using **FastAPI** as part of the FastAPI Internship Final Project.

It simulates a **Food Delivery Application** where users can browse menu items, place orders, manage a cart, and perform search, sorting, and pagination operations.



 Features

• REST APIs built with FastAPI  
• Pydantic data validation  
• CRUD operations for menu items  
• Cart and checkout workflow  
• Search functionality  
• Sorting of menu items  
• Pagination support  
• Combined browsing endpoint  
• API testing using Swagger UI  



Technologies Used

- Python
- FastAPI
- Pydantic
- Uvicorn
